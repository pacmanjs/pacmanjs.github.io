# Pac-Man Deluxe (HTML, CSS, JavaScript)

A lightweight, browser-based tribute to the legendary arcade game, Pac-Man, built entirely with HTML, CSS, and vanilla JavaScript. No frameworks or external dependencies required — just open `index.html` in your browser and start chomping\!

## 👻 Features

  * Classic arcade maze-chase gameplay
  * Multiple levels with increasingly complex mazes 🗺️
  * High score tracking to chase your personal best 🏆
  * Power Pellets to turn the tables on the ghosts\! 💪
  * Responsive design for desktop and mobile 📱
  * Retro pixel-art graphics and iconic sound effects 👾

## 🚀 Getting Started

### Play Locally

1.  Clone this repository:
    ```bash
    git clone https://github.com/pacmanjs/pacmanjs.github.io
    ```
2.  Navigate to the project folder:
    ```bash
    cd pacmanjs.github.io
    ```
3.  Open `index.html` in your web browser.

That’s it\! No build tools or server required.

## 🕹️ How to Play

  * Use the **Arrow Keys** or **WASD** to navigate Pac-Man through the maze.
  * Eat all the dots on the screen to advance to the next level.
  * Avoid the four ghosts: Blinky, Pinky, Inky, and Clyde\!
  * Eat a large **Power Pellet** to temporarily turn the ghosts blue, allowing you to eat them for extra points.
